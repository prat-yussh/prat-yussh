## About Me

I am a B.Tech Computer Science student from Centurion University, Odisha, with a strong interest in Cloud Computing, DevOps, Linux, Python, SQL, and automation.

I focus on learning through practical projects, clean documentation, and consistent hands-on practice. I am currently building my technical foundation in Cloud and DevOps while also exploring Data Analysis and Machine Learning to strengthen my problem-solving skills.

I enjoy understanding how systems work, improving my workflow with tools like Git and GitHub, and turning what I learn into projects that show practical implementation.
